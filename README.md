# Language Translator Website

A simple and responsive Language Translation Website built using HTML, CSS, and JavaScript.

## Features

- Translate text between multiple languages
- Clean and modern UI
- Swap languages instantly
- Clear input/output text
- Responsive design for mobile and desktop
- Uses MyMemory Translation API

## Technologies Used

- HTML5
- CSS3
- JavaScript
- MyMemory Translation API

## Supported Languages

- English
- Hindi
- Spanish
- French
- German
- Arabic
- Japanese
- Korean
- Malayalam
- Tamil
